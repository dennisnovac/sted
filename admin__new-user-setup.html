Default settings for new accounts

At this page you may configure default setting's for campaign creation page in simple and basic mode

DBM Field Name Description Value Option
override_ip IP used for sending test messages for all new accounts, example: "192.168.100.100" 1
override_helo Domain used for sending test messages for all new accounts, example: "mx.somedomain.com" 0
mailer_id Mailer ID used for sendings test messages, example: "1" 2
